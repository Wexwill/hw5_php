Многостраничный сайт с адаптивной версткой.
* Верстка с использованием преимущественно флекс-модели.
* Перечень страниц сайта (в т.ч. и ссылки в навигационном меню), а также содержимое внутренних страниц (кроме главной) хранятся в массиве в отдельном файле «pagesData.php». Отображение внутренних страниц реализовано через отдельный файл шаблона «page.php» и функцию получения данных по id, переданному через адресную строку.
* Создана функця для проверки отправленных данных формы: проверка телефона на соотв. международному формату (+375 ## ###-##-##); проверка адреса эл. почты; текст сообщения должен быть без тегов (удаление тегов если они будут переданы), более 5 символов и не превышать 250. Для проверки используются регулярные выражения.
* Если значения формы проходят проверку, то должна выполниться переадресация на страницу успешной отправки с благодарственным текстом, а через 10 секунд возвращение посетителя на страницу с контактами. Если форма не проходит проверку, посетителя отправляет обратно на страницу с контактами.
